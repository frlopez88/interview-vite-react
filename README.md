# React + Vite + Tailwind CSS Interview

This repository is designed for an interview focusing on basic Tailwind CSS configurations within a React and Vite environment.

## Overview

The purpose of this project is to demonstrate and evaluate fundamental skills utilizing Tailwind CSS.

## Interview Challenges

Please complete one challenge to demonstrate your Tailwind CSS skills.

## Instructions:
1. Clone the repo 
```git clone```

2. Install dependencies
```npm install```

3. Run the application
```npm run dev```

4. Pick one challenge and complete it.


### Challenge 1: Login Form Component (Freestyle)
Create a React component that renders a vertically and horizontally centered login form asking for a `username` and `password`.
- **Styling Requirements:** Freestyle! Use Tailwind CSS to make this form look nice, modern, and visually appealing. It's entirely up to you how to style the inputs, buttons and the form.

### Challenge 2: Light Mode Implementation
Create a Card Component with a title (your name) & description (your role).
- **Requirements:** Ensure the application is clearly styled for a **light theme** (e.g., white or light gray backgrounds with dark text).

### Challenge 3: Dark Mode Implementation
Create a Card Component with a title (your name) & description (your role).
- **Requirements:** Ensure the application is clearly styled for a **dark theme** (e.g., dark backgrounds with white text).
