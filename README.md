# React + Vite + Tailwind CSS Interview

This repository is designed for an interview focusing on basic Tailwind CSS configurations within a React and Vite environment.

## Overview

The purpose of this project is to demonstrate and evaluate fundamental skills in setting up, configuring, and utilizing Tailwind CSS.

## Interview Challenges

Please complete one challenge to demonstrate your Tailwind CSS skills.

### Challenge 1: Login Form Component (Freestyle)
Create a React component that renders a vertically and horizontally centered login form asking for a `username` and `password`.
- **Styling Requirements:** Freestyle! Use Tailwind CSS to make this form look nice, modern, and visually appealing. It's entirely up to you how to style the inputs, buttons, and the surrounding container/card.

### Challenge 2: Light Mode Implementation & Screenshot
Render the form you created in Challenge 1.
- **Requirements:** Ensure the application and the form are clearly styled for a **light theme** (e.g., white or light gray backgrounds with dark text). Once completed, take a screenshot of the entire project running in light mode and include it in the repository.

### Challenge 3: Dark Mode Implementation & Screenshot
Adapt the application to support dark mode using Tailwind's `dark:` variant.
- **Requirements:** Update the form from Challenge 1 so that it seamlessly transitions to a **dark theme** (e.g., dark slate backgrounds with white or light gray text). After making these adjustments, take a screenshot of the entire project running in dark mode and include it in the repository.
